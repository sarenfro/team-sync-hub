The Problem

Calendly works great — if you can afford it.

Tools like Calendly charge per user per month, which adds up fast for a volunteer-run student organization. For the UW Foster MBAA, that cost isn't worth it when the need is straightforward: students should be able to book time with exec council members without friction.

✦

The Solution

Team Sync Hub is a purpose-built, free scheduling platform for the UW Foster MBA community. Students can see council member availability and book meetings directly, no paid subscription required.

What it does

Built for how the MBAA actually works

📅

Self-serve scheduling

Students browse available times and book directly, no back-and-forth email chains required.

👥

Council member profiles

Each exec council member has their own availability and profile, so students know exactly who they're booking with.

🔒

Secure backend

Built on Supabase, so meeting data is stored securely with row-level access controls — no spreadsheet hacks.

💸

Completely free

No per-seat pricing, no trials, no paywalls. Designed specifically so student orgs don't need a budget line for scheduling.

⚡

Fast and lightweight

Powered by Vite and React, it loads instantly and works smoothly on both desktop and mobile.

🎓

Foster-specific

Scoped intentionally for the UW MBA cohort — not a generic tool trying to be everything to everyone.

Tech Stack

What it's built with

A modern frontend stack with a managed backend — chosen to minimize infrastructure overhead while keeping the codebase clean and maintainable.
